# 家計簿アプリ（Python）

## ■ 概要
Pythonで家計簿アプリを作成しました。  
CSVファイルを使用してデータを管理し、支出の記録・編集・集計・可視化ができるようにしています。

---

## ■ 機能

### ① データ入力
- 日付付きで支出を登録
- カテゴリは選択式（入力ミス防止）

### ② CRUD機能
- 追加
- 削除
- 更新
- 一覧表示

### ③ 集計機能
- 月ごとにカテゴリ別の支出を集計
- 支出の多い順にランキング表示

### ④ グラフ機能
- 指定した月の支出を円グラフで表示

---

## ■ 使用技術
- Python
- csv（データ保存）
- datetime（日付管理）
- matplotlib（グラフ表示）

---

## ■ 工夫した点
- カテゴリを選択式にすることで入力ミスを防止
- 不正な入力（文字・負の数）に対してエラー処理を実装
- CSVの読み込み・書き込みを関数で分離し、処理を整理
- 日付データを活用して月別集計機能を実装
- グラフ表示によりデータを視覚的に確認できるようにした
